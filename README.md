# Instalasi

1. Pastikan sudah terinstall Flutter SDK:<br>
   - Flutter Installation Guide
2. Clone repository:<br>
   - git clone <URL_REPOSITORY><br>
   - cd taskify
4. Install dependencies:<br>
   - flutter pub get
5. Jalankan aplikasi:<br>
   - flutter run
